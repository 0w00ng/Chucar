# Chucar
  신차/중고차 견적 어플리케이션

# 주요기능

  로그인 / 회원가입 / 결제
  견적서 요청 / 견적서 송신
  보낸 견적서 리스트 확인, 검색

# 적용기술

[COMMON]
  AWS(S3,EC2), 
  GIT, 
  RestAPI(Kakao,IamPort,Axios)

[F/E]
  ReactNative.
  NPM

[B/E]
  NodeJS,
  MySQL

# 실행방법

  npm install

  1st Terminal : npm start
  2nd Terminal : react-native run-ios
